# 🛒 Flipkart Customer Review Analyzer

An end-to-end Data Analytics project that analyzes customer reviews from an e-commerce dataset to extract meaningful insights using Python, Pandas, and Streamlit.

---

## 🌐 Live Demo  
👉 https://flipkart-review-analyzer-nfqbwayqydbrnitkbhizs9.streamlit.app/

---

## 📌 Overview

This project focuses on analyzing Flipkart product data to understand customer sentiment and identify commonly used words in reviews. It demonstrates practical skills in data cleaning, text processing, and interactive dashboard creation.

---

## 🚀 Features

- 🔍 **Word Frequency Analysis**  
  Identifies the most commonly used words in customer reviews.

- 😊 **Sentiment Analysis**  
  - Based on ratings (positive, negative, neutral)  
  - Based on keyword detection  

- 🧹 **Data Cleaning & Preprocessing**  
  - Handling missing values  
  - Converting data types  
  - Removing stopwords  

- 📊 **Interactive Dashboard**  
  Built using Streamlit with real-time visualization.

- 📂 **Default Dataset Support**  
  App works even without uploading a file.
